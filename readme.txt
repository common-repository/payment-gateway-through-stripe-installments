== Description ==
Use Payment Gateway throug Stripe Installments to accept credit card payments and allow your customers to split purchases over multiple billing statements. Available in Mexico only.

= Plugin for =
* Marketplaces
* Medical centres
* Other sites with goods&services

= How it works =
Payment Gateway through Stripe Installments allows customers to split the payment into parts according to one of the proposed plans for 3, 6, 9, 12, 18 and 24 months.

== Installation ==
= How to set up the plugin =
You should fill out the fields with the Secret key and Publishable key on the settings page. You can get it from your Stripe account (https://dashboard.stripe.com/dashboard). Important: Installment service must be activated in your stripe account. Available in Mexico only.
